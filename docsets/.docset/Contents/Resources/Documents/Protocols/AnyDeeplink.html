AnyDeeplink

A protocol that represents a deep link handler in an application. Types conforming to this protocol can identify and handle specific deep links.

  • canHandle(deeplink:)

    Determines whether a given URL can be handled by the conforming type.

  • handle(deeplink:navigator:)

    Handles the deep link and navigates to the appropriate screen.

    Note

    This function is asynchronous and may involve navigating between screens or handling background tasks.
